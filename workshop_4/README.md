# Задачи с четвертого(логически третьего) воркшопа


## 1. Обход бинарного дерева по уровням

Дан корень бинарного дерева. Нужно вернуть значения вершин в порядке обхода по уровням: слева направо, уровень за уровнем.

Результат должен быть представлен в виде списка списков, где каждый внутренний список содержит значения вершин одного уровня дерева.

**Требование:** решение должно использовать обход в ширину (BFS) с очередью и работать за `O(n)` по времени и `O(n)` по памяти, где `n` — количество вершин в дереве.

Источник задачи: https://leetcode.com/problems/binary-tree-level-order-traversal/description/

## 2. K-й наибольший элемент в массиве


Дан массив целых чисел `nums` и целое число `k`. Необходимо вернуть `k`-й наибольший элемент в массиве. Речь идёт о позиции в массиве, отсортированном по убыванию, а не о `k`-м уникальном элементе.

Пример:  
`nums = [3, 2, 1, 5, 6, 4], k = 2`  
Отсортированный по убыванию массив: `[6, 5, 4, 3, 2, 1]`  
2-й наибольший элемент = `5`.

**Требование:** решение должно использовать мин-кучу (`heapq`) фиксированного размера `k` и работать за `O(n * log k)` по времени и `O(k)` по памяти.

Источник задачи: https://leetcode.com/problems/kth-largest-element-in-an-array/description/
