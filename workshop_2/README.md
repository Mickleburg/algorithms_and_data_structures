# Условия задач первого воркшопа

## 1. Пересечение двух связанных списков

**Код решения**

`intersection.py`

**Условия:**

Даны два односвязных списка `headA` и `headB`.
Необходимо найти узел, с которого они пересекаются.
Если пересечения нет — вернуть `null`.

Пример: `listA = [4,1,8,4,5]`, `listB = [5,6,1,8,4,5]`

Результат: узел со значением `1`.


## 2. Сортировка стека через дополнительный стек

**Код решения**

`stack_sort.py`

**Условия**

Дан стек с целыми числами.
Необходимо отсортировать его так, чтобы наименьший элемент находился на вершине.

Разрешено использовать только *один дополнительный стек* и *одну вспомогательную переменную*.

Нельзя использовать массивы или другие структуры данных.
