# Задачи с третьего(логически второго) воршкопа

## 1. Длина наибольшей подстроки без повторяющихся символов

Дана строка `s`. Нужно найти длину самой длинной подстроки, в которой все символы уникальны.

**Требование:** решение должно работать за `O(n)`.

Источник задачи: https://leetcode.com/problems/longest-substring-without-repeating-characters/description/

---

## 2. K-й наименьший элемент в отсортированной матрице

Дана матрица `n x n`, где каждая строка и каждый столбец отсортированы по возрастанию.

Нужно найти `k`-й наименьший элемент.

**Требование:** решение должно работать за `O(n * log(max - min))` без дополнительной памяти.

Источник задачи: https://leetcode.com/problems/kth-smallest-element-in-a-sorted-matrix/description/
